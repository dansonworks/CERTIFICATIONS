# TryHackMe - Pre Security Learning Path

This repository contains my completion of the TryHackMe Pre Security learning path.

## Overview

Here is what I learned during this learning path:

## Networking Fundamentals
- What IP addresses are and how they function
- Basic understanding of DNS (Domain Name System)
- Introduction to HTTP and how web communication works
- Client-server model basics

## Linux Basics
- Navigating the Linux file system
- Basic command-line usage
- File and directory operations
- Introduction to permissions and user roles

## Web Fundamentals
- How websites and web applications work
- Basic structure of web requests and responses
- Overview of how browsers interact with servers

## Cybersecurity Fundamentals
- Core cybersecurity concepts and terminology
- Overview of common cyber threats and attack types
- Introduction to ethical hacking principles
- Basic security awareness practices

## Certification
This learning path was completed on TryHackMe and a certificate of completion is included in this repository.

## Purpose
This course was completed to build foundational knowledge in cybersecurity and prepare for more advanced certifications and hands-on home lab projects.
